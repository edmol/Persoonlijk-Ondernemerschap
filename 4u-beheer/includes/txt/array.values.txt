The following are post-initialized arrays of values for phpMyEdit forms.

To create a drop down that displays a set of 
possible values, consider the following field array:

$opts['fdd']['month'] = array(
  'default'    => '',
  'help|ACP'   => 'Select',
  'input'      => '',
  'maxlen'     => 2,
  'name'       => 'Month',
  'options'    => 'ACPVDFL',
  'required'   => false,
  'select'     => 'D',
  'sort'       => true,
  'values'     => array('', '01', '02', '03', '04', '05', '06', '07', '08', '09', '10', '11', '12')
);
